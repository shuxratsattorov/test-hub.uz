TEST HUB - TEST YARATISH VA ISHLASH UCHUN ENG QULAY PLATFORMA

Test Hub – bu foydalanuvchilarga test yaratish, ularni boshqalar bilan bo‘lishish va natijalarni kuzatish imkonini beruvchi platforma.

  • ✔ O‘zingizga mos mavzular bo‘icha testlar ishlashingiz
  • ✔ O‘zingiz test yaratishingiz, boshqalar bilan bolishishingiz
  • ✔ ID va parol orqali testlarga kirishingiz
  • ✔ Yaratilgan test natijalarini kuzatishingiz mumkin

Platformadan samarali foydalanish uchun quyidagi qo‘llanmaga amal qiling.

TEST HUB DAN FOYDALANISH QO‘LLANMASI:
  • Tayyor testlarni qayerdan topsam bo‘ladi?: Testlar bo‘limida kategoriyalar bo‘yicha o‘zingizga to‘gri keladigan testlarni tanlab ishlashigniz mumkin.
  • ID va parol orqali testni qayerdan topsam bo‘ladi?: Testlar bo‘limida o‘zingiz yoki boshqalar yaratgan testni ID va parol orqali testga kirib ishlashigniz mumkin.
  • Qanday qilib test yarataman?: Test qo‘shish bo‘limida, diqqat qiling "Asosiy ma‘lumot" sarlavhasi ostida formalarni to‘liq toldirib bo‘lib, "Savol qo‘shish" sarlavhasi ostida savol va javoblarni to'liq qo‘shib bolganingizdan so‘ng maxsus oyina ochiladi parolni kiritib saqlaganingizdan so‘ng, (notification) qo‘ng‘iroqchada sizga ID va parol taqdim etiladi.
  • Natijalarni qayerdan ko‘raman?: Natijalar bo‘limida deadline tugagandan so‘ng natijalar e‘lon qilinadi.
